HOW TO USE THIS WEBSITE
========================

FILES:
- index.html   → the page structure
- style.css    → all the styling (elegant fonts, colors, animations)
- script.js    → the fading gallery, gift card flip, and candle interaction
- images/      → put all your photos here

STEP 1 — ADD GALLERY PHOTOS (6-7 images)
-----------------------------------------
1. Put your photos inside the "images" folder.
2. Name them exactly: photo1.jpg, photo2.jpg, photo3.jpg ... photo7.jpg
   (If your files are .png or .jpeg, either rename them or open index.html
   and change the file extension in the src="" path to match.)
3. Each photo already has a quote under it in index.html, inside the
   <div class="gallery-quote">...</div> tag. Edit the quote text to
   whatever you like.
4. If you only have 6 photos, delete the 7th <div class="gallery-slide">...</div>
   block entirely from index.html.

STEP 2 — ADD GIFT CARD IMAGES (optional)
-----------------------------------------
1. Put up to 3 images in the "images" folder named:
   gift1.jpg, gift2.jpg, gift3.jpg
2. If you don't have images for the gift cards, that's fine — just
   leave them out and only the text message will show (the image
   tag will auto-hide if the file is missing).
3. Edit the <p class="gift-text">...</p> inside each card to write
   your own gift message (coupon, voucher, surprise, etc).

STEP 3 — PERSONALIZE THE MESSAGE
-----------------------------------------
Open index.html and find the "message-section" — edit the text inside
<p class="message-text"> to write your own personal note to Maira.

STEP 4 — OPEN THE WEBSITE
-----------------------------------------
Just double-click index.html to open it in any browser (Chrome, Edge,
Safari, etc). No internet/server needed — it works completely offline,
except the fonts which load from Google Fonts (needs internet for fonts
to display correctly, otherwise it falls back to a normal serif font).

STEP 5 — SHARE IT WITH MAIRA
-----------------------------------------
Once you're happy with it, you can:
- Zip the whole folder (index.html, style.css, script.js, images/) and send it
- Or host it for free using GitHub Pages / Netlify so she can open a real link
